# 基本面分析专家 (Fundamental Analysis)

## 概述

本Skill专注于股票基本面分析，通过深入研究公司的财务状况、行业地位、竞争优势、成长潜力等因素，评估资产的内在价值和投资潜力。

## 核心功能

### 1. 财务报表分析

**三大报表分析**
- 资产负债表分析（资产结构、负债水平、偿债能力）
- 利润表分析（盈利能力、成长性、费用控制）
- 现金流量表分析（现金流质量、运营效率）

**财务比率分析**
- 盈利能力：ROE、ROA、ROIC、毛利率、净利率
- 偿债能力：资产负债率、流动比率、速动比率
- 运营效率：周转率、存货周转、应收账款周转
- 成长能力：营收增长率、净利润增长率

### 2. 估值模型

**绝对估值法**
- DCF（现金流贴现模型）
- DDM（股利贴现模型）
- EVA（经济增加值）

**相对估值法**
- PE（市盈率）估值
- PB（市净率）估值
- PS（市销率）估值
- EV/EBITDA估值
- PEG估值

### 3. 行业分析

**行业研究框架**
- 市场规模与增长率
- 竞争格局（波特五力分析）
- 行业生命周期
- 技术发展趋势
- 政策法规影响

**竞争分析**
- 市场份额分析
- 竞争对手对比
- 护城河评估
- SWOT分析

### 4. 杜邦分析
- ROE三因素分解
- 财务杠杆分析
- 资产周转分析
- 利润率分析

## 财务指标详解

### 盈利能力指标

| 指标 | 公式 | 意义 |
|------|------|------|
| **ROE** | 净利润/净资产 | 股东权益回报率，核心盈利指标 |
| **ROA** | 净利润/总资产 | 资产利用效率 |
| **ROIC** | 税后经营利润/投入资本 | 投入资本回报率 |
| **毛利率** | (营收-成本)/营收 | 产品竞争力 |
| **净利率** | 净利润/营收 | 最终盈利能力 |

### 估值指标

| 指标 | 公式 | 适用场景 |
|------|------|---------|
| **PE** | 股价/每股收益 | 盈利稳定的公司 |
| **PB** | 股价/每股净资产 | 重资产行业 |
| **PS** | 市值/营收 | 亏损但高成长公司 |
| **PEG** | PE/盈利增长率 | 成长股估值 |
| **EV/EBITDA** | 企业价值/EBITDA | 跨公司、跨行业比较 |

### 杜邦分析公式

```
ROE = 净利润率 × 资产周转率 × 权益乘数

其中：
- 净利润率 = 净利润/营业收入
- 资产周转率 = 营业收入/总资产
- 权益乘数 = 总资产/净资产 = 1/(1-资产负债率)
```

## 行业分析框架

### 波特五力模型

1. **现有竞争者竞争强度**
2. **潜在进入者威胁**
3. **替代品威胁**
4. **供应商议价能力**
5. **客户议价能力**

### 行业生命周期

| 阶段 | 特征 | 投资策略 |
|------|------|---------|
| **导入期** | 高增长、高投入、低盈利 | 高风险高回报 |
| **成长期** | 快速增长、竞争加剧 | 优选龙头 |
| **成熟期** | 增长放缓、格局稳定 | 价值投资 |
| **衰退期** | 负增长、整合出清 | 避免或逆向 |

## 估值方法选择

### 不同行业适用估值方法

| 行业类型 | 推荐估值方法 | 原因 |
|---------|-------------|------|
| **金融** | PB、PE | 资产驱动、盈利稳定 |
| **科技** | PEG、PS、DCF | 高成长、盈利波动大 |
| **消费** | PE、DCF | 盈利稳定、现金流好 |
| **周期** | PB、EV/EBITDA | 盈利波动大 |
| **地产** | NAV、PE | 资产重、项目制 |
| **医药** | DCF、PE | 研发驱动、专利保护 |

## 分析流程

### 标准分析流程

1. **宏观环境扫描**
   - 经济周期判断
   - 货币政策分析
   - 行业政策梳理

2. **行业分析**
   - 市场规模测算
   - 竞争格局分析
   - 发展趋势判断

3. **公司分析**
   - 商业模式理解
   - 竞争优势评估
   - 财务数据分析

4. **估值分析**
   - 选择合适估值方法
   - 计算内在价值
   - 安全边际评估

5. **投资决策**
   - 综合评分
   - 风险提示
   - 投资建议

- 上游分析
- 中游分析
