# 投资风险决策分析专家 (Risk Decision Analysis)

## 概述

本Skill是一个综合投资风险决策分析框架，整合**激进型**、**保守型**、**中性型**三种风险分析视角，运用**博弈论**框架进行多轮风险博弈推演，对资产标的进行深度风险分析和决策评估。

**核心理念**：
1. **多视角分析**：从激进、保守、中性三种风险偏好角度全面审视投资机会
2. **博弈论推演**：运用博弈论框架分析市场参与者行为，推演不同情景下的最优决策
3. **风险优先**：优先识别和量化风险，在风险可控前提下追求收益
4. **动态决策**：根据市场变化和风险演化动态调整决策

## 分析框架

### 三维风险分析视角

#### 1. 激进型风险分析 (Aggressive Risk Analysis)
**核心思想**：追求高风险高收益，愿意承担较大波动换取超额回报

**分析重点**：
- 上行机会最大化
- 高Beta资产识别
- 杠杆使用策略
- 趋势动量捕捉
- 催化剂事件博弈

**风险偏好参数**：
- 最大回撤容忍度：30-50%
- 单笔止损：-10%至-15%
- 目标收益：+50%以上
- 仓位上限：单标的可达30%

#### 2. 保守型风险分析 (Conservative Risk Analysis)
**核心思想**：本金安全第一，追求稳定可持续的回报

**分析重点**：
- 下行风险最小化
- 安全边际评估
- 股息收益保障
- 防御性资产配置
- 黑天鹅事件防护

**风险偏好参数**：
- 最大回撤容忍度：10-15%
- 单笔止损：-5%至-8%
- 目标收益：+10%至+20%
- 仓位上限：单标的不超过10%

#### 3. 中性型风险分析 (Neutral Risk Analysis)
**核心思想**：平衡风险与收益，追求风险调整后的最优回报

**分析重点**：
- 风险收益比优化
- 夏普比率最大化
- 资产配置平衡
- 对冲策略设计
- 情景分析覆盖

**风险偏好参数**：
- 最大回撤容忍度：15-25%
- 单笔止损：-7%至-10%
- 目标收益：+20%至+35%
- 仓位上限：单标的15-20%

### 博弈论分析框架

#### 博弈参与者识别

| 参与者类型 | 特征 | 行为模式 | 影响权重 |
|-----------|------|---------|---------|
| **机构投资者** | 资金规模大、研究能力强 | 价值投资、长期持有 | 高 |
| **对冲基金** | 灵活、追求绝对收益 | 多空策略、事件驱动 | 高 |
| **散户投资者** | 情绪化、追涨杀跌 | 趋势跟随、羊群效应 | 中 |
| **公司内部人** | 信息优势 | 内幕交易、回购操作 | 高 |
| **做市商/量化** | 算法交易、流动性提供 | 高频交易、套利 | 中 |

#### 博弈情景推演

**第一轮：信息博弈**
- 各方信息优势分析
- 信息不对称程度评估
- 信息释放时机预测
- 信息冲击价格弹性

**第二轮：策略博弈**
- 各方策略空间分析
- 纳什均衡求解
- 占优策略识别
- 混合策略概率

**第三轮：动态博弈**
- 多期博弈树构建
- 逆向归纳求解
- 承诺与威胁分析
- 声誉效应评估

**第四轮：演化博弈**
- 策略演化路径
- 群体行为模式
- 市场结构变化
- 新均衡形成

### 多轮风险博弈分析

#### 第一轮：初始风险评估
**目标**：建立基础风险画像

**分析内容**：
1. **宏观风险扫描**
   - 经济周期位置判断
   - 货币政策风险评估
   - 地缘政治风险识别
   - 系统性风险压力测试

2. **行业风险分析**
   - 行业周期阶段
   - 政策监管风险
   - 技术替代威胁
   - 竞争格局变化

3. **公司特有风险**
   - 财务风险指标
   - 经营风险因素
   - 治理风险问题
   - 流动性风险评估

**输出**：风险矩阵初稿

#### 第二轮：情景构建与压力测试
**目标**：识别极端风险情景

**分析内容**：
1. **基准情景** (概率50%)
   - 假设条件
   - 预期收益
   - 风险水平

2. **乐观情景** (概率25%)
   - 催化剂事件
   - 上行空间
   - 实现路径

3. **悲观情景** (概率25%)
   - 风险触发因素
   - 下行幅度
   - 持续时间

4. **极端情景** (概率5-10%)
   - 黑天鹅事件
   - 系统性冲击
   - 流动性危机

**输出**：情景分析报告

#### 第三轮：对手方行为推演
**目标**：分析市场参与者行为对价格的影响

**分析内容**：
1. **机构持仓分析**
   - 机构持股比例变化
   - 重仓机构行为预测
   - 调仓换股影响

2. **做空势力评估**
   - 做空比例变化
   - 轧空风险分析
   - 做空成本评估

3. **散户情绪分析**
   - 情绪指标监控
   - 羊群效应评估
   - 极端情绪识别

4. **内部人行为**
   - 增减持动态
   - 回购计划
   - 股权激励影响

**输出**：市场博弈态势图

#### 第四轮：动态风险调整
**目标**：建立动态风险管理机制

**分析内容**：
1. **风险预算分配**
   - 总体风险预算
   - 各情景风险分配
   - 动态调整规则

2. **触发机制设计**
   - 加仓触发条件
   - 减仓触发条件
   - 清仓触发条件

3. **对冲策略选择**
   - 期权对冲
   - 期货对冲
   - 跨资产对冲

**输出**：动态风险管理方案

## 风险量化模型

### 综合风险评分体系

| 风险维度 | 权重 | 激进视角 | 保守视角 | 中性视角 |
|---------|------|---------|---------|---------|
| 市场风险 | 20% | 高容忍 | 低容忍 | 中容忍 |
| 信用风险 | 15% | 中容忍 | 低容忍 | 中容忍 |
| 流动性风险 | 15% | 中容忍 | 低容忍 | 中容忍 |
| 操作风险 | 10% | 高容忍 | 低容忍 | 中容忍 |
| 系统性风险 | 20% | 中容忍 | 低容忍 | 中容忍 |
| 尾部风险 | 20% | 高容忍 | 极低容忍 | 低容忍 |

### 风险调整收益计算

**激进型风险调整收益**
```
RAR_Aggressive = 预期收益 / (下行风险 × 0.7)
```

**保守型风险调整收益**
```
RAR_Conservative = 预期收益 / (下行风险 × 1.5)
```

**中性型风险调整收益**
```
RAR_Neutral = 预期收益 / 下行风险
```

### 风险价值 (VaR) 与条件风险价值 (CVaR)

**VaR计算**
```
VaR_95 = μ - 1.645 × σ
VaR_99 = μ - 2.326 × σ
```

**CVaR计算**
```
CVaR_95 = E[X | X ≤ VaR_95]
```

### 博弈论最优策略选择

**占优策略识别**
- 无论对手如何选择都是最优的策略
- 适用于信息优势明显的场景

**纳什均衡策略**
- 各方都无法通过单方面改变策略而获得更好结果
- 适用于市场均衡状态

**混合策略概率**
- 根据概率分布随机选择策略
- 适用于高度不确定环境

## 风险管理建议

### 激进型风险管理

**核心原则**：高风险高回报，严格止损保护本金

**具体措施**：
1. **仓位管理**
   - 单标的最大仓位30%
   - 高相关性标的组合不超过50%
   - 保留20%现金应对机会

2. **止损策略**
   - 硬止损：-10%至-15%
   - 移动止损：盈利20%后止损上移至成本价
   - 时间止损：持仓30天未达预期减半

3. **杠杆使用**
   - 最大杠杆2倍
   - 杠杆仓位不超过总仓位30%
   - 严格监控保证金水平

### 保守型风险管理

**核心原则**：本金安全第一，追求稳定收益

**具体措施**：
1. **仓位管理**
   - 单标的最大仓位10%
   - 分散投资至少10个标的
   - 保留30%现金作为缓冲

2. **止损策略**
   - 硬止损：-5%至-8%
   - 分批建仓，每次不超过计划仓位1/3
   - 股息再投资降低风险

3. **防御配置**
   - 防御性行业占比不低于40%
   - 高股息标的占比不低于30%
   - 债券/现金占比不低于20%

### 中性型风险管理

**核心原则**：风险收益平衡，动态调整配置

**具体措施**：
1. **仓位管理**
   - 单标的最大仓位15-20%
   - 根据市场环境动态调整
   - 保留15-25%现金

2. **对冲策略**
   - 使用期权保护下行风险
   - 行业多空配对
   - 跨市场分散

3. **动态平衡**
   - 定期再平衡（季度/半年）
   - 根据风险预算调整仓位
   - 情景变化时及时调整

## 评分标准

### 各视角评分体系

| 维度 | 激进视角权重 | 保守视角权重 | 中性视角权重 |
|------|-------------|-------------|-------------|
| 上行潜力 | 30% | 10% | 20% |
| 下行风险 | 15% | 35% | 25% |
| 风险收益比 | 15% | 25% | 30% |
| 流动性 | 10% | 15% | 10% |
| 催化剂 | 20% | 5% | 10% |
| 安全边际 | 10% | 10% | 5% |

### 综合评分计算

```
综合评分 = 激进评分 × 0.3 + 保守评分 × 0.3 + 中性评分 × 0.4
```

### 投资建议生成

| 综合评分 | 投资建议 | 风险等级 |
|---------|---------|---------|
| 90-100 | 强烈买入 | 激进型适合 |
| 80-89 | 买入 | 激进/中性适合 |
| 70-79 | 增持 | 中性适合 |
| 60-69 | 持有 | 保守型适合 |
| 50-59 | 减持 | 风险偏高 |
| 40-49 | 卖出 | 风险过高 |
| <40 | 强烈卖出/做空 | 极高风险 |
