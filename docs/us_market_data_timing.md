# 美股行情数据获取时效逻辑

## 核心原则
**分析时间点 → 确定应使用的数据日期**

## 美股交易时间（北京时间）
- **夏令时**：21:30 - 次日 04:00
- **冬令时**：22:30 - 次日 05:00

## 数据时效性判断逻辑

### 场景1：北京时间当日 04:00/05:00 之前
- 美股前一交易日尚未收盘
- **应使用数据**：T-2 日收盘数据（前天）
- **示例**：北京时间 3月4日 03:00 分析 → 使用 3月2日 数据

### 场景2：北京时间当日 04:00/05:00 之后
- 美股前一交易日已收盘
- **应使用数据**：T-1 日收盘数据（昨天）
- **示例**：北京时间 3月4日 10:00 分析 → 使用 3月3日 数据 ✅

### 场景3：北京时间当日 21:30/22:30 之后
- 美股当日已开盘
- **应使用数据**：实时盘中数据 或 T-1 日收盘数据（根据需求）
- **示例**：北京时间 3月4日 22:00 分析 → 可使用 3月4日 实时数据

## 快速判断公式

```
当前北京时间 = X

if X < 04:00 (或 05:00 冬令时):
    数据日期 = 今天 - 2天
elif 04:00 <= X < 21:30 (或 22:30 冬令时):
    数据日期 = 今天 - 1天  ✅ 最常见场景
else:
    数据日期 = 今天（实时）或 今天 - 1天
```

## 实际案例

| 分析时间（北京时间） | 美股状态 | 应使用数据日期 | 说明 |
|-------------------|---------|--------------|------|
| 3月4日 03:00 | 3月3日未收盘 | 3月2日 | 凌晨4点前用前天数据 |
| 3月4日 10:00 | 3月3日已收盘 | **3月3日** | 凌晨4点后用昨天数据 |
| 3月4日 15:00 | 3月3日已收盘 | **3月3日** | 凌晨4点后用昨天数据 |
| 3月4日 22:00 | 3月4日交易中 | 3月4日实时 或 3月3日 | 根据需求选择 |

## 常见错误

❌ **错误**：北京时间 3月4日 10:00 分析，使用 3月2日 数据  
✅ **正确**：北京时间 3月4日 10:00 分析，使用 3月3日 数据

原因：3月3日美股在北京时间 3月4日 凌晨4点已收盘，10点时数据已可用。

## 数据获取检查清单

执行美股分析前，必须确认：
- [ ] 当前北京时间是几点？
- [ ] 美股前一交易日是否已收盘？
- [ ] 应使用哪一天的收盘数据？
- [ ] 获取的数据日期是否正确？
