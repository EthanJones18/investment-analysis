# 图表形态识别指南

## 支撑与阻力

### 支撑位 (Support)
价格下跌时遇到的买盘力量，阻止价格继续下跌的水平。

**特征**:
- 价格多次触及后反弹
- 成交量在支撑位附近放大
- 突破后支撑位变为阻力位

**识别方法**:
1. 找局部最低点
2. 合并相近的价格水平
3. 统计触碰次数

### 阻力位 (Resistance)
价格上涨时遇到的卖盘力量，阻止价格继续上涨的水平。

**特征**:
- 价格多次触及后回落
- 成交量在阻力位附近放大
- 突破后阻力位变为支撑位

### 交易策略

| 场景 | 策略 |
|------|------|
| 价格接近支撑位 | 考虑买入，止损设在支撑位下方 |
| 价格突破支撑位 | 卖出，支撑位变阻力 |
| 价格接近阻力位 | 考虑卖出或减仓 |
| 价格突破阻力位 | 买入，阻力位变支撑 |

## 趋势线

### 上升趋势线
连接两个或多个低点，斜率为正。

**画法**:
1. 找两个明显的低点
2. 用直线连接
3. 确保中间没有价格跌破趋势线

**交易信号**:
- 价格触及趋势线反弹：买入
- 价格跌破趋势线：趋势可能反转，卖出

### 下降趋势线
连接两个或多个高点，斜率为负。

**画法**:
1. 找两个明显的高点
2. 用直线连接
3. 确保中间没有价格突破趋势线

**交易信号**:
- 价格触及趋势线回落：卖出
- 价格突破趋势线：趋势可能反转，买入

### 趋势线强度

| 因素 | 说明 |
|------|------|
| 触碰次数 | 触碰越多，趋势线越有效 |
| 时间跨度 | 时间越长，趋势线越重要 |
| 角度 | 45度角的趋势线最稳定 |
| 成交量 | 突破时成交量放大，确认有效 |

## K线形态

### 单根K线形态

#### 锤子线 (Hammer)
**形态**: 小实体，长下影线，短上影线（或无）
**位置**: 下跌趋势末端
**含义**: 看涨反转信号

#### 倒锤子线 (Inverted Hammer)
**形态**: 小实体，长上影线，短下影线（或无）
**位置**: 下跌趋势末端
**含义**: 看涨反转信号（需次日确认）

#### 十字星 (Doji)
**形态**: 开盘价≈收盘价，上下影线可长可短
**位置**: 任何位置
**含义**: 市场犹豫，可能反转或继续

#### 纺锤线 (Spinning Top)
**形态**: 小实体，上下影线较长
**位置**: 趋势中
**含义**: 多空力量均衡，可能反转

### 双根K线形态

#### 看涨吞没 (Bullish Engulfing)
**形态**: 
- 第一根：阴线，小实体
- 第二根：阳线，大实体，完全包住第一根

**位置**: 下跌趋势末端
**含义**: 强烈看涨反转

#### 看跌吞没 (Bearish Engulfing)
**形态**:
- 第一根：阳线，小实体
- 第二根：阴线，大实体，完全包住第一根

**位置**: 上涨趋势末端
**含义**: 强烈看跌反转

#### 刺透形态 (Piercing Pattern)
**形态**:
- 第一根：阴线
- 第二根：阳线，开盘价低于前低，收盘价深入前实体50%以上

**位置**: 下跌趋势
**含义**: 看涨反转

#### 乌云盖顶 (Dark Cloud Cover)
**形态**:
- 第一根：阳线
- 第二根：阴线，开盘价高于前高，收盘价深入前实体50%以上

**位置**: 上涨趋势
**含义**: 看跌反转

### 三根K线形态

#### 早晨之星 (Morning Star)
**形态**:
- 第一根：大阴线
- 第二根：小实体（可星线），跳空低开
- 第三根：大阳线，深入第一根实体

**位置**: 下跌趋势末端
**含义**: 强烈看涨反转

#### 黄昏之星 (Evening Star)
**形态**:
- 第一根：大阳线
- 第二根：小实体（可星线），跳空高开
- 第三根：大阴线，深入第一根实体

**位置**: 上涨趋势末端
**含义**: 强烈看跌反转

#### 三白兵 (Three White Soldiers)
**形态**: 连续三根阳线，每根开盘价在前一根实体内，收盘价创新高
**位置**: 下跌趋势后
**含义**: 强烈看涨

#### 三乌鸦 (Three Black Crows)
**形态**: 连续三根阴线，每根开盘价在前一根实体内，收盘价创新低
**位置**: 上涨趋势后
**含义**: 强烈看跌

## 经典图表形态

### 反转形态

#### 头肩顶 (Head and Shoulders)
**形态**: 左肩-头-右肩，颈线连接两个低点
**成交量**: 头部成交量小于左肩
**目标价**: 头部到颈线的距离，向下投射

#### 头肩底 (Inverse Head and Shoulders)
**形态**: 左肩-头-右肩（倒置），颈线连接两个高点
**成交量**: 右肩成交量放大
**目标价**: 头部到颈线的距离，向上投射

#### 双顶 (Double Top)
**形态**: 两个相近的高点，中间有回调
**颈线**: 连接两个回调低点
**目标价**: 顶到颈线的距离，向下投射

#### 双底 (Double Bottom)
**形态**: 两个相近的低点，中间有反弹
**颈线**: 连接两个反弹高点
**目标价**: 底到颈线的距离，向上投射

### 持续形态

#### 三角形 (Triangle)

**对称三角形**:
- 上轨下降，下轨上升
- 突破方向不确定，跟随原趋势概率大

**上升三角形**:
- 上轨水平，下轨上升
- 看涨形态，向上突破

**下降三角形**:
- 上轨下降，下轨水平
- 看跌形态，向下突破

#### 旗形 (Flag)

**上涨旗形**:
- 快速上涨后，小幅回调，通道向下倾斜
- 看涨持续形态

**下跌旗形**:
- 快速下跌后，小幅反弹，通道向上倾斜
- 看跌持续形态

#### 楔形 (Wedge)

**上升楔形**:
- 两轨都向上，上轨角度小于下轨
- 看跌反转形态

**下降楔形**:
- 两轨都向下，下轨角度小于上轨
- 看涨反转形态

## 形态识别注意事项

1. **等待确认**: 形态完成后等待价格突破确认
2. **成交量**: 突破时成交量放大，确认有效性
3. **时间跨度**: 形态形成时间越长，突破后力度越大
4. **假突破**: 设置止损，防范假突破
5. **结合使用**: 形态+指标+成交量，提高准确性
