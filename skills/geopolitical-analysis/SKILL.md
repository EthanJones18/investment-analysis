# 地缘政治分析专家 (Geopolitical Analysis)

## 概述

本Skill专注于从地缘政治角度分析资产，通过收集和解读全球重大地缘政治事件、冲突、政策变化等信息，评估其对资产价格的潜在影响。

**核心原则**：
1. **实时监控**：持续跟踪全球地缘政治动态
2. **事件分级**：根据严重程度对事件进行分类
3. **情景推演**：基于历史模式推演可能的发展路径
4. **资产映射**：建立事件与资产的关联影响模型
5. **及时预警**：对重大风险/机会发出强烈信号

## 核心功能

### 1. 全球事件监控（必须执行）
- GDELT全球事件数据库实时监控
- ACLED武装冲突数据追踪
- 新闻源地缘政治事件提取
- 社交媒体地缘政治信号监控

### 2. 事件分类与评估（必须执行）
- CAMEO事件编码分类
- Goldstein冲突合作分数计算
- 事件严重程度分级（CRITICAL/HIGH/MEDIUM/LOW/INFO）
- 事件类型识别（冲突/制裁/外交/经济/网络/贸易等）

### 3. 历史情景分析（必须执行）
- 相似历史事件回溯
- 事件发展路径推演
- 资产价格历史反应分析
- 风险传导链条建模

### 4. 资产影响评估
- 直接暴露分析（业务所在地）
- 间接暴露分析（供应链、市场）
- 行业暴露分析
- 情绪传导分析

### 5. 预警系统
- 实时风险评级
- 机会/风险信号生成
- 多情景概率评估
- 关键指标监控

## 地缘政治事件分类体系

### 事件类型

| 类型 | 代码 | 描述 | 典型资产影响 |
|------|------|------|-------------|
| **军事冲突** | CONFLICT | 战争、军事打击、边境冲突 | 能源、军工、黄金、避险资产 |
| **经济制裁** | SANCTION | 贸易禁运、金融制裁、技术封锁 | 被制裁国资产、替代供应商 |
| **外交紧张** | DIPLOMATIC | 断交、驱逐外交官、外交抗议 | 双边贸易相关资产 |
| **经济政策** | ECONOMIC | 货币政策、财政政策、产业政策 | 汇率、债券、股市 |
| **恐怖主义** | TERRORISM | 恐怖袭击、安全威胁 | 航空、旅游、保险 |
| **网络攻击** | CYBER | 网络战、数据泄露、关键基础设施攻击 | 科技、金融、基础设施 |
| **贸易争端** | TRADE | 关税、贸易战、WTO争端 | 进出口企业、大宗商品 |
| **选举政治** | ELECTION | 政权更迭、政策转向、政治危机 | 本国资产、相关产业链 |

### 严重程度分级

| 级别 | 代码 | 描述 | 市场反应 | 响应时间 |
|------|------|------|---------|---------|
| **CRITICAL** | 5 | 战争爆发、大规模冲突 | 剧烈波动 | 立即 |
| **HIGH** | 4 | 制裁实施、军事对峙 | 显著波动 | 数小时 |
| **MEDIUM** | 3 | 外交紧张、抗议活动 | 中度波动 | 数天 |
| **LOW** | 2 | 政策变化、谈判进展 | 轻度波动 | 数周 |
| **INFO** | 1 | 一般新闻、背景信息 | 无显著影响 | 持续监控 |

## 使用方法

### 完整地缘政治分析（推荐）

```bash
python scripts/comprehensive_analysis.py \
  --symbol BABA \
  --watch-regions east_asia,middle_east \
  --days 30 \
  --output ./output
```

### 单独模块分析

#### 全球事件监控

```bash
python scripts/geopolitical_monitor.py \
  --countries CHN,USA,RUS,IRN,ISR \
  --days 7 \
  --output ./output
```

#### 历史情景分析

```bash
python scripts/scenario_analysis.py \
  --symbol BABA \
  --event-type conflict \
  --historical-events "2022-02-24,2014-03-18" \
  --output ./output
```

```bash
python scripts/asset_impact.py \
  --symbol BABA \
  --events-file ./output/geopolitical_events.json \
  --output ./output
```

## 事件-资产映射模型

### 映射维度

| 维度 | 说明 | 示例 |
|------|------|------|
| **地理暴露** | 资产在事件地区的业务 | 中概股在中美紧张时的风险 |
| **供应链暴露** | 关键供应链节点 | 芯片短缺对科技股的冲击 |
| **市场暴露** | 目标市场依赖度 | 俄乌冲突对欧洲能源股的影响 |
| **行业暴露** | 行业敏感性 | 军工股在冲突升级时的机会 |
| **情绪暴露** | 市场情绪传导 | 避险情绪对黄金的影响 |

### 影响评分

影响分数范围：-1.0（极度负面）到 +1.0（极度正面）

| 分数区间 | 影响程度 | 投资建议 |
|---------|---------|---------|
| 0.7 ~ 1.0 | 强烈正面 | 强烈买入 |
| 0.3 ~ 0.7 | 正面 | 买入 |
| -0.3 ~ 0.3 | 中性 | 观望 |
| -0.7 ~ -0.3 | 负面 | 减仓/对冲 |
| -1.0 ~ -0.7 | 强烈负面 | 强烈卖出/避险 |

## 情景分析框架

### 情景构建

1. **基准情景（Base Case）**
   - 概率：40-50%
   - 描述：最可能的发展路径
   - 资产影响：温和

2. **乐观情景（Bull Case）**
   - 概率：20-30%
   - 描述：紧张局势缓解
   - 资产影响：正面

3. **悲观情景（Bear Case）**
   - 概率：20-30%
   - 描述：冲突升级
   - 资产影响：负面

4. **极端情景（Tail Risk）**
   - 概率：<10%
   - 描述：黑天鹅事件
   - 资产影响：剧烈

### 关键指标监控

| 指标类型 | 具体指标 | 预警阈值 |
|---------|---------|---------|
| **冲突强度** | 军事活动频率 | >3σ |
| **外交信号** | 双边关系分数 | <-0.5 |
| **市场情绪** | 恐慌指数 | >30 |
| **资金流向** | 避险资产流入 | >2σ |
| **政策变化** | 制裁/关税公告 | 新事件 |

## 预警系统

### 预警级别

| 级别 | 颜色 | 触发条件 | 响应措施 |
|------|------|---------|---------|
| **红色** | 🔴 | CRITICAL事件 + 资产直接暴露 | 立即避险/对冲 |
| **橙色** | 🟠 | HIGH事件 + 资产间接暴露 | 减仓/调整仓位 |
| **黄色** | 🟡 | MEDIUM事件 + 行业暴露 | 加强监控 |
| **蓝色** | 🔵 | 新事件但影响有限 | 记录观察 |

### 预警推送

- 实时推送：CRITICAL/HIGH级别事件
- 日报：每日事件汇总
- 周报：趋势分析和情景更新
- 专题报告：重大事件深度分析

## 分析流程

### 标准分析流程

1. **事件发现**
   - 多源数据监控
   - 事件去重和关联
   - 初步分类和分级

2. **深度分析**
   - 事件背景研究
   - 历史相似事件对比
   - 发展路径推演

3. **影响评估**
   - 资产暴露分析
   - 传导链条建模
   - 量化影响评分

4. **情景构建**
   - 多情景概率评估
   - 关键指标识别
   - 触发条件设定

5. **预警生成**
   - 风险/机会信号
   - 投资建议
   - 对冲策略
