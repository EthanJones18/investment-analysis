# 财务指标详解

## 盈利能力指标

### ROE (净资产收益率)
**公式**: ROE = 净利润 / 净资产

**意义**:
- 衡量股东权益的回报率
- 是巴菲特最看重的指标之一
- 持续高ROE（>15%）表明公司有竞争优势

**判断标准**:
| ROE范围 | 评价 |
|---------|------|
| >20% | 优秀 |
| 15-20% | 良好 |
| 10-15% | 一般 |
| <10% | 较差 |

**注意事项**:
- 高杠杆可能推高ROE，需结合杜邦分析
- 一次性收益可能扭曲ROE
- 应观察5-10年趋势

### ROA (总资产收益率)
**公式**: ROA = 净利润 / 总资产

**意义**:
- 衡量资产利用效率
- 不受资本结构影响

**判断标准**:
| ROA范围 | 评价 |
|---------|------|
| >10% | 优秀 |
| 5-10% | 良好 |
| <5% | 一般 |

### ROIC (投入资本回报率)
**公式**: ROIC = 税后经营利润 / (股东权益 + 有息负债)

**意义**:
- 衡量所有投入资本的回报
- 最适合跨公司比较

**判断标准**:
| ROIC范围 | 评价 |
|---------|------|
| >15% | 优秀，有护城河 |
| 10-15% | 良好 |
| <10% | 一般，价值毁灭 |

### 毛利率
**公式**: 毛利率 = (营业收入 - 营业成本) / 营业收入

**意义**:
- 反映产品竞争力和定价权
- 高毛利率通常意味着品牌优势或技术壁垒

**行业差异**:
| 行业 | 典型毛利率 |
|------|-----------|
| 软件/互联网 | 70-90% |
| 医药 | 60-80% |
| 消费品 | 40-60% |
| 制造业 | 20-40% |
| 零售 | 10-20% |

### 净利率
**公式**: 净利率 = 净利润 / 营业收入

**意义**:
- 最终盈利能力
- 反映费用控制能力

## 偿债能力指标

### 资产负债率
**公式**: 资产负债率 = 总负债 / 总资产

**意义**:
- 衡量财务杠杆水平
- 过高可能面临偿债风险

**判断标准**:
| 比率范围 | 评价 |
|---------|------|
| <40% | 保守 |
| 40-60% | 适中 |
| 60-70% | 较高 |
| >70% | 高风险 |

**行业差异**:
- 银行、地产：高杠杆正常
- 科技、消费：低杠杆更优

### 流动比率
**公式**: 流动比率 = 流动资产 / 流动负债

**意义**:
- 衡量短期偿债能力

**判断标准**:
| 比率范围 | 评价 |
|---------|------|
| >2 | 优秀 |
| 1.5-2 | 良好 |
| 1-1.5 | 一般 |
| <1 | 危险 |

### 速动比率
**公式**: 速动比率 = (流动资产 - 存货) / 流动负债

**意义**:
- 更严格的短期偿债指标
- 剔除变现能力差的存货

**判断标准**:
| 比率范围 | 评价 |
|---------|------|
| >1 | 优秀 |
| 0.8-1 | 良好 |
| <0.8 | 需关注 |

### 利息保障倍数
**公式**: 利息保障倍数 = EBIT / 利息费用

**意义**:
- 衡量偿付利息能力

**判断标准**:
| 倍数范围 | 评价 |
|---------|------|
| >5 | 优秀 |
| 3-5 | 良好 |
| 1-3 | 一般 |
| <1 | 危险，无法覆盖利息 |

## 运营效率指标

### 总资产周转率
**公式**: 总资产周转率 = 营业收入 / 总资产

**意义**:
- 衡量资产利用效率
- 与利润率共同决定ROA

**行业差异**:
| 行业类型 | 典型周转率 |
|---------|-----------|
| 零售业 | 2-3 |
| 制造业 | 1-2 |
| 重资产行业 | 0.3-0.5 |

### 存货周转率
**公式**: 存货周转率 = 营业成本 / 平均存货

**意义**:
- 衡量存货管理效率
- 过低可能积压，过高可能缺货

### 应收账款周转率
**公式**: 应收账款周转率 = 营业收入 / 平均应收账款

**意义**:
- 衡量回款效率
- 过低可能坏账风险

---

## 成长能力指标

### 营收增长率
**公式**: 营收增长率 = (本期营收 - 上期营收) / 上期营收

**判断标准**:
| 增长率 | 评价 |
|--------|------|
| >30% | 高速增长 |
| 15-30% | 快速增长 |
| 5-15% | 稳定增长 |
| <5% | 增长乏力 |

### 净利润增长率
**公式**: 净利润增长率 = (本期净利润 - 上期净利润) / 上期净利润

**注意事项**:
- 应关注扣非净利润增长
- 一次性收益可能扭曲增长

---

## 现金流指标

### 经营现金流/净利润
**公式**: 经营现金流/净利润 = 经营活动现金流 / 净利润

**意义**:
- 衡量盈利质量
- >1说明盈利有现金支撑

### 自由现金流
**公式**: 自由现金流 = 经营现金流 - 资本支出

**意义**:
- 可分配给股东和债权人的现金
- 持续为正说明公司有造血能力

---

## 估值指标

### 市盈率 (PE)
**公式**: PE = 股价 / 每股收益

**意义**:
- 最常用的估值指标
- 反映市场对公司未来增长的预期

**判断标准**:
| PE范围 | 评价 |
|--------|------|
| <10 | 可能被低估 |
| 10-20 | 合理区间 |
| 20-30 | 成长溢价 |
| >30 | 高估值，需高增长支撑 |

### 市净率 (PB)
**公式**: PB = 股价 / 每股净资产

**意义**:
- 适用于重资产行业
- <1可能被低估

### PEG
**公式**: PEG = PE / 盈利增长率

**意义**:
- 成长股估值指标
- <1被认为低估

---

## 综合评估框架

### 优秀公司的财务特征

1. **盈利能力**
   - ROE > 15%，持续5年以上
   - ROIC > 10%
   - 毛利率稳定或提升

2. **成长能力**
   - 营收增长率 > 10%
   - 净利润增长率 > 营收增长率

3. **偿债能力**
   - 资产负债率 < 60%
   - 流动比率 > 1.5

4. **现金流**
   - 经营现金流/净利润 > 1
   - 自由现金流持续为正

5. **运营效率**
   - 周转率稳定或提升
   - 费用率下降
