# 技术指标参考手册

## 趋势指标

### 1. 移动平均线 (Moving Average)

#### 简单移动平均线 (SMA)
**公式**: SMA = (P1 + P2 + ... + Pn) / n

**参数**:
- 短期: 5, 10, 20日
- 中期: 30, 60日
- 长期: 120, 250日

**交易信号**:
- 价格上穿MA：买入
- 价格下穿MA：卖出
- 短期MA上穿长期MA（金叉）：买入
- 短期MA下穿长期MA（死叉）：卖出

#### 指数移动平均线 (EMA)
**公式**: EMA_today = P_today * k + EMA_yesterday * (1-k), k=2/(n+1)

**特点**: 对近期价格赋予更高权重，反应更灵敏

### 2. MACD (指数平滑异同平均线)

**公式**:
- DIF = EMA(12) - EMA(26)
- DEA = EMA(DIF, 9)
- MACD柱 = (DIF - DEA) * 2

**交易信号**:
- DIF上穿DEA（金叉）：买入
- DIF下穿DEA（死叉）：卖出
- MACD柱由负转正：买入信号增强
- MACD柱由正转负：卖出信号增强
- 背离：价格新高但MACD未新高，可能反转

### 3. ADX (平均趋向指数)

**公式**:
- +DI = 100 * (+DM的EMA) / ATR
- -DI = 100 * (-DM的EMA) / ATR
- DX = 100 * |+DI - (-DI)| / (+DI + (-DI))
- ADX = DX的EMA

**解读**:
- ADX < 20：趋势弱，市场震荡
- ADX > 25：趋势强
- ADX > 40：趋势极强
- +DI > -DI：上升趋势
- +DI < -DI：下降趋势

## 动量指标

### 1. RSI (相对强弱指数)

**公式**: RSI = 100 - (100 / (1 + RS))
- RS = 平均上涨幅度 / 平均下跌幅度

**参数**: 默认14日

**交易信号**:
- RSI > 70：超买，考虑卖出
- RSI < 30：超卖，考虑买入
- RSI背离：价格新高但RSI未新高，可能反转

### 2. KDJ (随机指标)

**公式**:
- RSV = (Close - Low_n) / (High_n - Low_n) * 100
- K = RSV的EMA
- D = K的EMA
- J = 3K - 2D

**参数**: n=9, m1=3, m2=3

**交易信号**:
- K上穿D（金叉）：买入
- K下穿D（死叉）：卖出
- K > 80：超买区
- K < 20：超卖区
- J > 100：严重超买
- J < 0：严重超卖

### 3. 威廉指标 (Williams %R)

**公式**: %R = (High_n - Close) / (High_n - Low_n) * -100

**交易信号**:
- %R > -20：超买
- %R < -80：超卖

## 波动率指标

### 1. 布林带 (Bollinger Bands)

**公式**:
- 中轨 = MA(20)
- 上轨 = MA(20) + 2 * STD(20)
- 下轨 = MA(20) - 2 * STD(20)

**交易信号**:
- 价格触及下轨后反弹：买入
- 价格触及上轨后回落：卖出
- 带宽收窄（挤压）：大行情即将开始
- 带宽扩张：波动加大

### 2. ATR (平均真实波幅)

**公式**:
- TR = max(High - Low, |High - Close_prev|, |Low - Close_prev|)
- ATR = TR的EMA

**用途**:
- 设置止损位：2-3倍ATR
- 衡量市场波动性
- 仓位管理

## 成交量指标

### 1. OBV (能量潮)

**公式**:
- 收盘价上涨：OBV = OBV_prev + Volume
- 收盘价下跌：OBV = OBV_prev - Volume
- 收盘价持平：OBV = OBV_prev

**解读**:
- OBV上升，价格上升：确认上涨趋势
- OBV下降，价格下降：确认下跌趋势
- OBV与价格背离：可能反转

### 2. VMA (成交量移动平均线)

**用途**:
- 成交量突破VMA：确认价格突破
- 缩量回调：健康调整
- 放量滞涨：警惕顶部

## 多指标组合策略

### 策略1：趋势跟踪
**指标**: MA + MACD + ADX

**买入条件**:
1. 价格 > MA(20) > MA(50)
2. MACD金叉且MACD柱 > 0
3. ADX > 25且+DI > -DI

**卖出条件**:
1. 价格 < MA(20)
2. MACD死叉

### 策略2：均值回归
**指标**: 布林带 + RSI

**买入条件**:
1. 价格触及布林带下轨
2. RSI < 30

**卖出条件**:
1. 价格触及布林带上轨
2. RSI > 70

### 策略3：动量突破
**指标**: KDJ + 成交量

**买入条件**:
1. KDJ金叉
2. K从20以下向上突破
3. 成交量放大

**卖出条件**:
1. KDJ死叉
2. K从80以上向下突破

## 参数优化建议

### 不同市场参数调整

| 市场类型 | 趋势指标周期 | 动量指标周期 | 特点 |
|---------|-------------|-------------|------|
| 股票 | 20-50日 | 14日 | 趋势性较强 |
| 期货 | 10-20日 | 9-14日 | 波动大，反应要快 |
| 加密货币 | 10-20日 | 7-10日 | 24小时交易，周期缩短 |
| 外汇 | 20-50日 | 14日 | 趋势性强，但假突破多 |

### 避免过度优化

1. **样本外测试**: 用历史数据训练，用近期数据验证
2. **参数稳健性**: 参数小幅变化不应导致结果大幅变化
3. **交易成本**: 考虑滑点和手续费
4. **市场变化**: 策略需要定期调整
