# 交易策略设计指南

## 概述

交易策略是投资计划的核心执行部分，它将投资主题转化为具体的买卖操作。一个好的交易策略应该明确入场条件、仓位管理规则和出场策略。

## 入场策略

### 入场条件设计

#### 必要条件清单
- [ ] 投资决策综合评分 ≥ 70分
- [ ] 价格进入目标区间
- [ ] 技术形态符合要求
- [ ] 风险收益比 ≥ 1:3
- [ ] 流动性充足
- [ ] 市场环境适宜

#### 入场时机类型

| 时机类型 | 适用场景 | 优点 | 缺点 | 风险等级 |
|---------|---------|------|------|---------|
| 突破入场 | 技术形态突破 | 趋势确认 | 假突破风险 | 中 |
| 回调入场 | 上涨趋势回调 | 成本较低 | 可能继续下跌 | 中低 |
| 分批入场 | 不确定性高 | 分散风险 | 可能错过机会 | 低 |
| 事件驱动 | 催化剂事件 | 爆发力强 | 时间不确定 | 高 |
| 定投入场 | 长期投资 | 平滑成本 | 短期效果差 | 低 |

### 入场价格区间

#### 价格区间确定方法

**技术分析法**
```
支撑位 = 近期低点 / 均线位置 / 前期平台
阻力位 = 近期高点 / 历史高点 / 心理关口

理想入场价 = 支撑位附近
最高入场价 = 阻力位下方5%
```

**估值分析法**
```
合理估值区间 = [合理PE下限 × EPS, 合理PE上限 × EPS]
理想入场价 = 合理估值区间下限
最高入场价 = 合理估值区间上限
```

**综合定价**
```
入场价格区间 = 技术分析区间 ∩ 估值分析区间
理想入场价 = min(技术支撑位, 估值下限)
最高入场价 = max(技术阻力位, 估值上限) × 0.95
```

## 仓位管理策略

### 初始仓位计算

#### 风险比例法 (推荐)
```
初始仓位 = 可承受亏损金额 ÷ (入场价 - 止损价)

示例:
账户资金: 100万
单笔风险限额: 2% = 2万
入场价: 100元
止损价: 90元

初始仓位 = 20,000 ÷ (100 - 90) = 2,000股
初始资金 = 2,000 × 100 = 20万 (20%)
```

#### 固定比例法
```
初始仓位 = 账户总资产 × 固定比例

保守型: 5%
中性型: 10%
激进型: 15-20%
```

#### 凯利公式
```
f* = (bp - q) / b

其中:
b = 赔率 = 预期收益 / 预期亏损
p = 胜率
q = 1 - p

示例:
预期收益: 30%
预期亏损: 10%
胜率: 60%

b = 30% / 10% = 3
f* = (3 × 0.6 - 0.4) / 3 = 0.47 (47%)

实际应用: 使用半凯利或1/4凯利
半凯利: 23.5%
1/4凯利: 11.75%
```

### 加仓策略

#### 金字塔加仓法 (推荐)

| 加仓次数 | 仓位比例 | 触发条件 | 累计仓位 | 适用场景 |
|---------|---------|---------|---------|---------|
| 首次 | 30% | 达到入场条件 | 30% | 趋势初期 |
| 第二次 | 20% | 盈利10%后 | 50% | 趋势确认 |
| 第三次 | 15% | 盈利20%后 | 65% | 趋势延续 |
| 第四次 | 10% | 盈利30%后 | 75% | 趋势强劲 |

**优点**: 顺势加仓，风险可控
**缺点**: 初期仓位较轻

#### 倒金字塔加仓法 (谨慎使用)

| 加仓次数 | 仓位比例 | 触发条件 | 累计仓位 | 适用场景 |
|---------|---------|---------|---------|---------|
| 首次 | 10% | 达到入场条件 | 10% | 试探性建仓 |
| 第二次 | 15% | 下跌5%后 | 25% | 摊低成本 |
| 第三次 | 20% | 下跌10%后 | 45% | 继续摊低 |
| 第四次 | 30% | 下跌15%后 | 75% | 重仓抄底 |

**优点**: 成本摊低效果明显
**缺点**: 逆势加仓，风险较高
**适用**: 价值投资、长期看好

#### 固定比例加仓法

| 加仓次数 | 仓位比例 | 触发条件 | 累计仓位 |
|---------|---------|---------|---------|
| 首次 | 20% | 达到入场条件 | 20% |
| 第二次 | 20% | 达到加仓条件 | 40% |
| 第三次 | 20% | 达到加仓条件 | 60% |
| 第四次 | 20% | 达到加仓条件 | 80% |

**优点**: 简单易执行
**缺点**: 不够灵活

### 减仓策略

#### 止盈减仓

**分批止盈**
| 盈利水平 | 减仓比例 | 剩余仓位 | 操作理由 |
|---------|---------|---------|---------|
| +15% | 减仓25% | 75% | 锁定部分利润 |
| +25% | 减仓25% | 50% | 回收本金 |
| +35% | 减仓25% | 25% | 保留利润头寸 |
| +50% | 清仓 | 0% | 目标达成 |

**移动止盈减仓**
```
盈利20%: 减仓20%，止损上移至成本价
盈利30%: 再减仓20%，止损上移至盈利10%
盈利50%: 再减仓30%，止损上移至盈利25%
盈利100%: 清仓或保留极轻仓
```

#### 风险减仓

| 风险信号 | 减仓比例 | 操作理由 |
|---------|---------|---------|
| 基本面恶化 | 减仓50% | 降低风险敞口 |
| 技术破位 | 减仓30% | 趋势可能反转 |
| 宏观风险 | 减仓30% | 系统性风险 |
| 达到止损 | 清仓 | 控制损失 |
| 波动率异常 | 减仓20% | 不确定性增加 |

## 出场策略

### 止盈策略

#### 固定比例止盈
- **保守型**: +15%、+20%、+25%
- **中性型**: +25%、+35%、+50%
- **激进型**: +50%、+100%、+200%

**优点**: 简单明确，易于执行
**缺点**: 可能错失更大涨幅

#### 移动止盈

**跟踪止损**
```
盈利10%: 止损上移至成本价 (保本)
盈利20%: 止损上移至盈利5%
盈利30%: 止损上移至盈利10%
盈利50%: 止损上移至盈利20%
盈利100%: 止损上移至盈利50%
```

**回撤止盈**
```
从最高点回撤10%: 减仓50%
从最高点回撤15%: 清仓
```

#### 技术止盈

| 技术信号 | 操作 | 说明 |
|---------|------|------|
| 到达重要阻力位 | 减仓 | 可能回调 |
| RSI > 80 | 减仓 | 超买 |
| 成交量异常放大 | 减仓 | 可能顶部 |
| 形态完成 | 清仓 | 目标达成 |
| 均线死叉 | 清仓 | 趋势反转 |

### 止损策略

#### 固定比例止损

| 风险偏好 | 止损比例 | 适用场景 |
|---------|---------|---------|
| 保守型 | -5%至-8% | 低波动标的 |
| 中性型 | -8%至-12% | 中等波动 |
| 激进型 | -12%至-15% | 高波动标的 |

#### 技术止损

| 技术信号 | 止损位置 | 说明 |
|---------|---------|------|
| 跌破关键支撑 | 支撑位下方3% | 支撑变压力 |
| 趋势线破位 | 趋势线下方 | 趋势可能反转 |
| 形态失败 | 形态最低点 | 突破失败 |
| 均线死叉 | 短期均线下方 | 短期走弱 |

#### 时间止损
```
持仓周期: 根据投资周期设定
- 短线: 5-10天
- 中线: 1-3个月
- 长线: 6-12个月

时间止损规则:
- 超过预期时间未达目标: 评估退出
- 避免资金占用成本
- 机会成本考虑
```

#### 波动率止损
```
止损价 = 入场价 - 2 × ATR(14)

ATR(14): 14日平均真实波幅

优点: 自适应波动率变化
缺点: 波动率扩大时止损距离变大
```

#### 基本面止损
```
触发条件:
- 业绩不及预期
- 行业政策重大变化
- 公司治理问题
- 竞争优势丧失

操作: 立即清仓，不犹豫
```

## 策略组合示例

### 保守型策略组合

```
入场: 回调至支撑位，分批建仓
初始仓位: 5%
加仓: 每下跌5%加仓5%，最多20%
止损: -7%或跌破关键支撑
止盈: +15%减仓50%，+25%清仓
```

### 中性型策略组合

```
入场: 趋势确认后，金字塔加仓
初始仓位: 10%
加仓: 盈利10%加仓5%，最多20%
止损: -10%或移动止损
止盈: +25%减仓25%，+35%减仓25%，+50%清仓
```

### 激进型策略组合

```
入场: 突破入场，快速建仓
初始仓位: 15%
加仓: 盈利10%加仓10%，最多30%
止损: -12%或技术止损
止盈: +50%减仓30%，+100%减仓30%，跟踪止盈
```

## 策略评估

### 评估指标

| 指标 | 说明 | 优秀标准 |
|------|------|---------|
| 胜率 | 盈利交易占比 | > 50% |
| 盈亏比 | 平均盈利/平均亏损 | > 2:1 |
| 夏普比率 | 风险调整收益 | > 1.5 |
| 最大回撤 | 最大资金回落 | < 20% |
| 平均持仓 | 平均持仓时间 | 符合预期 |

### 策略优化

1. **回测验证**: 历史数据回测
2. **参数优化**: 找到最优参数
3. **样本外测试**: 避免过拟合
4. **实盘验证**: 小资金实盘测试
5. **持续优化**: 根据市场变化调整
