# 投资交易专家 (Investment Trading)

## 概述

本Skill是一个投资交易执行框架，根据投资决策分析结果，制定详细的交易计划。涵盖仓位管理、入场出场策略、风险控制、交易执行等具体可操作的方案，帮助投资者将投资决策转化为实际交易行动。

**核心理念**：
1. **计划先行**：每笔交易前制定详细计划，避免冲动交易
2. **风险控制**：优先保护本金，严格控制单笔亏损
3. **纪律执行**：严格执行交易计划，避免情绪化操作
4. **持续优化**：通过交易日志复盘，不断优化交易系统

## 仓位管理方法

### 固定比例法
```
单笔仓位 = 账户总资产 × 固定比例(如5%)
```

### 风险比例法
```
单笔仓位 = 可承受亏损金额 ÷ (入场价 - 止损价)
```

### 凯利公式
```
f* = (bp - q) / b
其中: b=赔率, p=胜率, q=败率
```

### 金字塔加仓
| 加仓次数 | 仓位比例 | 触发条件 |
|---------|---------|---------|
| 首次 | 30% | 达到入场条件 |
| 第二次 | 20% | 盈利5%后 |
| 第三次 | 15% | 盈利10%后 |
| 第四次 | 10% | 盈利15%后 |

## 出场策略

### 止盈策略

**1. 固定比例止盈**
- 目标收益: +15%、+25%、+30%
- 达到目标后分批止盈

**2. 移动止盈**
- 盈利10%后，止损上移至成本价
- 盈利20%后，止损上移至盈利10%位置
- 跟踪止损: 回撤10%止盈

**3. 技术止盈**
- 到达重要阻力位
- 技术指标超买
- 成交量异常

### 止损策略

**1. 固定比例止损**
- 硬止损: -5%、-7%、-8%
- 根据波动率调整

**2. 技术止损**
- 跌破关键支撑位
- 趋势线破位
- 形态失败

**3. 时间止损**
- 持仓超过预期时间未达目标
- 避免资金占用成本

## 风险管理规则

### 单笔交易风险
- 最大亏损不超过账户2%
- 止损位必须明确设置
- 不补仓摊低成本（除非原计划）

### 单日风险控制
- 单日最大亏损不超过账户5%
- 连续3笔亏损停止交易
- 情绪失控立即停止

### 总体风险控制
- 最大回撤不超过账户20%
- 单一标的不超过账户20%
- 现金比例不低于10%

## 交易心理管理

### 常见交易心理陷阱
1. **FOMO（错失恐惧）**: 追涨杀跌
2. **锚定效应**: 过度关注成本价
3. **确认偏误**: 只寻找支持自己观点的信息
4. **过度自信**: 高估自己的判断能力
5. **损失厌恶**: 过早止盈，过晚止损

### 心理管理技巧
- 交易前冥想/深呼吸
- 设置交易规则并严格执行
- 定期复盘，客观分析
- 保持交易日志，追踪情绪
- 适当休息，避免过度交易
